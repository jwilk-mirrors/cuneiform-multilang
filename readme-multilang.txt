This is a multi-language (modified) version of cuneiform currently mantained by Andrei Borovsky (anb@symmetrica.net).
Additional features (regarding original cuneiform 0.6.0) include
-recognizing mixed languages texts. For example, if you want to recognize text containing Russian and French fragments, select -l rus_fra key.
-forsing input text layout representation as a single column (helps to keep layout in some cases). Use -c1 key to enable this.
